# boston summer survival villa

A personal pixel-style website inspired by:
- watching Obsession
- Love Island season 8
- hating UFC on the TV
- Boston summer heat
- the new AC unit
- both of you applying to jobs

## Customize it

Open `index.html` and edit the letter text.

Open `index.js` and change the inside-joke buttons or messages.

Open `style.css` and change the colors at the top.

## Run it

Open `index.html` in a browser, or upload the folder to GitHub Pages.
